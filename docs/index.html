REST API Center

A REST API Center with authentication created in CodeIgniter 3

About this REST API in CodeIgniter

This REST API server is created in the PHP framework CodeIgniter 3. It is made for future projects that want to use a REST API.

This REST API server uses the following libraries:

  • Ion Auth (See onGitHub).
  • CodeIgniter Rest Controller (See onGitHub).
  • Format class (author: @softwarespot)
  • API Auth (author: Aaron van den Berg)

Authentications

The REST API server is set by default with the two following authentications; user account and API-Key.
